## 六阶段生产流程 / Six-Stage Production Process

### 中文

| 阶段 | 名称 | 核心作用 |
|:---|:---|:---|
| **阶段一** | 全剧灵魂与顶层设计 | 解构素材，输出世界观、角色档案、关系矩阵 |
| **阶段二** | 多维度风格决策 | 受众画像分析，输出2-3套视觉方案供选择 |
| **阶段三** | 视觉规格总结 | 将选定风格转化为AI生图/生视频的技术协议 |
| **阶段四** | 资产多态库 | 人物/道具/场景的中文提示词矩阵 |
| **阶段五** | 单集重构分析 | 将原文改写为漫剧专用执行剧本 |
| **阶段六** | 分镜执行表 | 生成可直接驱动AI视频的工业级执行清单 |

### English

| Stage | Name | Core Function |
|:---|:---|:---|
| **Stage 1** | Core Concept | Deconstruct materials, output world view, character profiles, relationship matrix |
| **Stage 2** | Visual Strategy | Audience profiling, output 2-3 visual schemes for selection |
| **Stage 3** | Visual Specification | Convert selected style to AI image/video generation technical protocol |
| **Stage 4** | Asset Library | Chinese prompt matrix for characters/props/scenes |
| **Stage 5** | Script Engineering | Rewrite original text into motion comic execution script |
| **Stage 6** | Production Map | Generate industrial execution list for AI video generation |
